Git

Git como herramienta para la gestión de la configuración del software.

Git tiene diversos usos y facilidades tanto para individuos como para grupos de trabajo completos.
Puede crear ramas completas para que se modifiquen de forma asincrona con otros miembros de equipo, además de guardar de forma segura las distintas versiones de un programa.
Si se usa un versionador remoto incluso se salva un proyecto de ser perdido al perderse la unidad de almacenamiento local
